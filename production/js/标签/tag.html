(1)初识HTML

html不是编程语言，它只是一种文本标记语言，特点：简易性、可扩展性、平台无关性、通用性；

(2)一个标准html页面的模板

<!doctype html>
	<html>
		<head>
			<!--申明当前页面的编码集（中文编码gb2312，国际编码UTF-8）-->
			<meta http-equiv="Content-Type" content="text/html";charset="utf-8">
			<!--申明当前页面的三要素-->
			<title>HTML模板</title>

			<meta name="KeyWords" content="关键词,关键词">
			<meta name="Description" content="">
		</head>
		<body>
			<!--页面的主体内容-->
		</body>
	</html>

(13)iframe标签

用法说明：iframe 元素会创建包含另外一个文档的内联框架（即行内框架）
iframe常用的几个属性：frameborder——规定是否显示框架周围的边框；height——高度；width——宽度；src——规定在iframe中显示的文档的URL

(14)pre标签

用法说明：pre 元素可定义预格式化的文本，被包围在 pre 元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体。 pre 标签的一个常见应用就是用来表示计算机的源代码。显示效果参见上面的HTML模板效果
